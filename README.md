## Lab Questions

### In plain English describe the function and use of a multiplexer.

A multiplexer takes multiple inputs and a switch value, and uses the switch value to allow one of the inputs to pass through. The analogy we used in class was a railway switch, where multiple tracks converge into one and only one train can pass through the junction at a time.

### In plain English describe the function and use of a demultiplexer.

A demultiplexer takes a single input and a switch value, and uses the switch value to send the input to one of multiple possible outputs.

### What other uses might these circuits have? (Think Shannon’s)

Muxes can act as fundamental blocks, sort of like NANDS. That is, a function can be built using only muxes. Shannon’s expansion works by breaking functions into their constituent parts in a way that can be easily represented with muxes alone.
